Coding Flash Attention from scratch using triton and pytorch.

## Setup environment
* Clone the repository and Go to `triton-flash-attn` directory.
```bash
git clone https://github.com/eljandoubi/triton-flash-attn.git && cd triton-flash-attn
```

* Build environment.
```bash
make build
```

## Clean environment
```bash
make clean
```
